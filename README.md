# Countro


Countro is login Register based app using firebase with Facebook and Google sign along with navigation controller and auto layout and progressive bar that uses Public API to retrieve a list of all countries.

  
### Tech

Countro uses a number of cocopods to work properly:

* Alamofire - for networking
* Firebase - for authentication 
* FBSDKLoginKit - Login Using Facebook
* GoogleSignIn - Sigin using Google
* SwiftyJSON - Easy JSON handler




### Installation

Only need to make sure that all the pods are installed correctly 

```sh
$ cd directoryFolder
$ pod init //copy and paste what in the podfile then
$ pod install
```
